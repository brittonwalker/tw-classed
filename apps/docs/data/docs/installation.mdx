## Install it

```bash
# With npm
npm install tw-classed

# With yarn
yarn add tw-classed

# With pnpm
pnpm install tw-classed
```

## Import it

<>import `classed` from `tw-classed`</>;

```tsx
import classed from "tw-classed";
```

## Use it

Use the `classed` function to create a component and style it with Tailwind

```tsx
import classed from "tw-classed";

const Button = classed(
  "button",
  "bg-blue-500 text-white font-bold py-2 px-4 rounded-md",
  "hover:bg-blue-700"
);
```
